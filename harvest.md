##### gotoxy函数
1. 通过gotoxy函数可以设置光标到文本屏幕的指定位置，如果在 Turbo C 或 Borland C 中使用 gotoxy 函数，就需要在文件头部添加包含头文件 conio.h代码。
2. 而在 Visual C++ 或 GCC 中使用 gotoxy 函数，就需要把上面自定义的 gotoxy 函数写在自己的程序中。同时注意Windows.h头文件。
***修改文字颜色的函数同理。***
##### \b
`printf("请选择[1 2 3]:[ ]\b\b");        //\b为退格，使得光标处于[]中间`
使用2个`\b`:使用1个光标停在“]”出，使用两个才能停在“[]”里面。
